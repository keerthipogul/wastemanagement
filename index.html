Waste Management System

This project uses a Convolutional Neural Network (CNN) to classify waste images into Wet, Dry, and E-Waste categories.

About the Project

Waste management is important to maintain a clean environment. This project uses deep learning techniques to identify different types of waste automatically using image classification.

Waste Types

  • Wet Waste – Food and organic waste
  • Dry Waste – Plastic and paper waste
  • E-Waste – Electronic devices

Algorithm

1. Collect waste images (Wet, Dry, E-Waste)
2. Preprocess images (Resize and Normalize)
3. Train CNN model using TensorFlow
4. Input new waste image
5. Predict waste type

Python Code


import tensorflow as tf
from tensorflow.keras import layers, models

model = models.Sequential()

model.add(layers.Conv2D(32,(3,3),activation='relu',input_shape=(128,128,3)))
model.add(layers.MaxPooling2D(2,2))

model.add(layers.Conv2D(64,(3,3),activation='relu'))
model.add(layers.MaxPooling2D(2,2))

model.add(layers.Conv2D(128,(3,3),activation='relu'))
model.add(layers.MaxPooling2D(2,2))

model.add(layers.Flatten())
model.add(layers.Dense(128,activation='relu'))
model.add(layers.Dense(3,activation='softmax'))

Project Result

The trained CNN model successfully predicts the category of waste from input images.
