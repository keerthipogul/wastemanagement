Waste Management System

This project classifies waste into Wet, Dry, and E-Waste using CNN.

About the Project

Waste management is important to keep the environment clean. This project uses a Convolutional Neural Network (CNN) model to classify waste images into different categories.

Waste Types

  • Wet Waste – Food and organic waste
  • Dry Waste – Plastic and paper waste
  • E-Waste – Electronic devices

Algorithm

1. Collect waste images (Wet, Dry, E-Waste)
2. Preprocess images (Resize and Normalize)
3. Train CNN model using TensorFlow
4. Input new image
5. Predict waste type

Technology Used

Python, TensorFlow, CNN, HTML, GitHub

Project Result

The trained CNN model successfully predicts the category of waste from input images.
